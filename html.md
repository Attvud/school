# HTML5 тэги

- **html** - обьясняет компьютеру, что мы будем писать HTML код
- **head** - внутри него служебная информация сайта и подключение CSS стилей
- **body** - внутры него все видимое содержимое страницы
- **h1-h6** - заголовки страницы
- **p** - один абзац текста
- **br** - переносит текст на новую строку
- **a** - ссылка
- **img** - картинка
- **button** - кнопка
- **hr** - линия-разделитель
- **ul** - обычный список
- **ol** - нумерованный список
- **li** - один элемент списка
