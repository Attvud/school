# HTML5 тэги

**html** - обьясняет компьютеру, что мы будем писать HTML код
**head** - внутри него слуюебная информация сайта и подключение CSS стилей
**body** - внутры него все видимое содержимое страницы
**p** - один абзац текста
**br** - перенести текст на новую строку
**a** - ссылка
**img** - картинка
**button** - кнопка
**hr** - линия-разделитель
**ul** - обычный список
**ol** - нумерованный список
**li** - один элемент списка
