# CSS3 свойства

- **border: 1px solid black** - позволяет одновременно установить толщину, стиль и цвет границы вокруг элемента
- **background-color** - цвет фона элемента

---

Свойства для текста

- **color: green** - цвет
- **font-size: 16px** - размер
- **font-weight: bold** - толщина/жирность шрифта
- **font-family: Arial** - выбрать тип/семейство шрифтов
- **text-decoration** - оформление текста
